# 快捷键

```
创建虚拟桌面:			Win + Ctrl+ D	(MacOS: Ctrl+Up)
切换虚拟桌面:			Win + Ctrl + ←/→	(MacOS: Ctrl + ←/→)
关闭虚拟桌面:			Win + Ctrl + F4	(MacOS: Option+单击关闭)

切换到桌面:			
	Win + M
	Win + D (再按恢复)
临时显示桌面:		    Win + ,	(Comma逗号)	(MacOS: Command+F3)
当前窗口最小化:	   Win + ↓	(最大化: Win + ↑, 多按试效果) 
当前窗口靠左侧:	   Win + ←	(靠左侧: Win + →) 
关闭当前窗口:			Win + W
显示开始菜单:			Ctrl + Esc

显示运行的所有程序:
	Ctrl + Alt + Tab		(MacOS: Ctrl+Tab)
	Ctrl + Tab
	Win + Tab
打开任务管理器
	Ctrl + Shift + Esc		(MacOS: Command+Option+Escape)
	Ctrl + Shift + Del, 再点击'任务管理器'
打开我的电脑:		 Win + E
锁屏:				Win + L
截图自动保存至桌面:	Win + PrintScreen

打开放大镜:		  Win + +/-

快速重命名:		F2
删除:			
	Ctrl D
	Del
	Shift + Del (永久删除)

新建文件夹:		Ctrl + Shift + N
创建快捷方式:		拖动时按 Ctrl + Shift
```
