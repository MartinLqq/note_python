# 数据库考虑点

1. 是否开源
2. 关系型 / 非关系型
3. 是否需要预先创建数据库、表、字段
4. 存储数据是否需要预先设定格式
5. 存储的数据类型
6. 持久化存储:  支持 / 不支持
7. 外键:  支持 / 不支持
8. 索引:  支持 / 不支持
9. 事务:  支持 / 不支持
10. 触发器:  支持 / 不支持
11. 并发:  支持 / 不支持
12. 数据库存储的原理
13. 使用场景
